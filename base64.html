Apa itu Base64?

Base64 adalah skema pengkodean biner-ke-teks yang merepresentasikan data biner dalam format string ASCII. Ini sering digunakan dalam pengembangan web untuk menyematkan data gambar ke dalam file CSS atau HTML, atau mentransmisikan data mentah melalui protokol yang hanya mendukung teks.
